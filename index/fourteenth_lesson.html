Приемы в CSS

Многоуровневое меню

Основной идеей создания многоуровневого меню является использование вложенных друг в друга списков «ul». Главный список – это список классом «ul.menu», где элементы списка «li», в котором находится еще один список (подменю) с классом «ul.submenu». для того, чтобы при наведении на элемент списка «menu» появлялось подменю, мы задаем для «ul.submenu» свойство «display: none». И при псевдоклассе «:hover» список «ul.submenu» устанавливало свойство «display:block».

Бургер-крестик анимация

Основной идеей такой анимации является создания элемента (помощью свойства «border» создается рамка бургера), для которого создается псевдоэлемент «::before» (одна черточка). И для этого псевдоэлемента с помощью свойства «box-shadow» создается другая черточка (вторая черточка). И, наконец, третья черточка создается с помощью псевдоэлемнта «::after».

Для того, чтобы происходила анимация при наведении на бургер, необходимо для верхней и нижней черточек использовать свойства «transform: translate» «transform: rotate», при этом средняя черточка прячется с помощью отмены свойства «box-shadow».

Обрезание блока по координатам «clip-path»

Такой прием реализуется (например, картинка) благодаря свойству «clip-path: polygon(x1 y1, x2 y2…(%))»

Выезжающая боковая панель

Основной идеей такого подхода является задаем свойства для меню, когда оно еще не выехало – «position: fixed», «top:0», «bottom» и смещаем его на всю ширину меню, оставляя где-то 10рх в пределах экрана. Когда меню выехало задаем «left:0». Ну и в конце, задаем псевдокласс «:hover» на само меню, и теперь при наведении мыши на край меню, который остался в пределах экрана, меню выезжает на экран.

Эффект «paralax»

Здесь создается контейнер, которому задаются такие свойства: «height: 100vh» «position: relative» «overflow-x: hidden» «overflow-y: auto/scroll» «perspective:1px»

После этого с помощью псевдоэлемента создается фон контейнеру такими свойствами: «position: absolute» «left:0» «right:0» «top:0» «bottom: 0» «background-image» «transform: translate(-5px) scale(n)»

Но для того, чтобы фон ушел на самый задний план необходимо сверх лежащему контенту задать «z-index: 1».
