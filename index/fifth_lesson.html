Свойство "text-transform" имеет следующие значения:

  • uppercase (все буквы заглавные)
  • lowercase (все буквы маленькие)
  • capitalize (начало предложения с большой буквы)

псевдокласс «:hover» отвечает за действия при наведении мыши на элемент и записывается следующим образом:

.button:hover{действия, которые совершаются при наведении на элемент «button»}

Подключение шрифтов

  1. Скачивается с «Google fonts» необходимый шрифт (несколько файлов для разного типа жирности и наклонения)
  2. Создается папка «fonts», в которой будут находится скаченные файлы
  3. В этой же папке создается файл, например, с названием «fonts.css», в которой подключаются все необходимые шрифты
  4. Затем этот файл «fonts.css» подключатся в *.html

Подключение видео на страницу

Подключение видео на страницу с помощью тега «iframe». Для этого правой кнопкой мыши выбираем «копировать HTML код» на необходимом видео и вставляем его в *.html.


box-shadow (в нем есть такое свойство, как «inset», делает тень внутри блока)



Использование свойства «display: inline-block» для выравнивания элементов на странице:

  1. Задается элементам, которых нужно выровнять свойство «display: inline-block»
  2. Родителю этих элементов задается свойство «text-align: justify»
  3. Но есть один момент свойство «text-align: justify» не растягивает по ширине последнюю строку. Поэтому данному родителю добавляется еще одна пустая строка
  4. Поскольку строка по умолчание уже имеет какую-то высоту, то для этого этому же родителю задают такие 2 свойства: «line-height: 0» и «font-size: 0»
