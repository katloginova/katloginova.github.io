CSS свойства

Язык CSS состоит из селекторов и свойств. Селекторы указывают на то, к каким элементам применять стили, а свойства — на то, как именно отображать элементы.

Существует огромное количество CSS-свойств, которые влияют практически на все аспекты отображения элементов. Причём каждому свойству соответствует определённый набор значений. Некоторые значения задаются с помощью текстовых констант, другие с помощью цифровых значений.

В данном уроке рассматривалось такое понятие, как inline-стили, т.е. стили, которые описываются HTML документе (style="свойство: значение;").

Раньше разметка страницы HTML производилась с помощью таблиц. Создание таблицы осуществляется с помощью тега table в состав, которого входят теги thead, tbody, tfoot. За строки отвечает тег tr, а за столбцы (т.е. ячейки) - td. Есть еще тег th - ячейки с заголовками (жирный шрифт) таблицы.

За объединение столбцов отвечает атрибут colspan="кол-во столбцов", который прописывается в теге td. За объединение строк - атрибут rowspan="кол-во строк" также в теге td.

Свойство Значение Описание
border px solid color рамка элемента (задается толщина, форма и цвет)
width px(pt, %) ширина элемента
height px(pt, %) высота элемента
text-align center(left, right, justify) выравнивает текст по центру(слева, справа, по ширине)
font-weight normal(bold) жирность шрифта
font-style normal(italic) стиль шрифта
background-color black(#000) цвет фона
