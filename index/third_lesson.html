Мое первое использование CSS стилей для создания сайта

Назначение CSS – отделять то, что задает внешний вид страницы, от ее содержания. Если документ создан только с использованием HTML, то в нем определяется не только каждый элемент, но и способ его отображения. Если же подключены каскадные таблицы стилей, то HTML описывает только очередность объектов. А за все их свойства отвечает CSS.

Для того, чтобы задать стиль какому-нибудь элементу необходимо к нему как-то обратиться. И для этого существуют несколько способов обращения:

  • * - для всей страницы;
  • .classname - для элементов, имеющих данный класс;
  • #id - уникальный код (используется в коде документа только один раз);
  • tagname - для тегов

Часто используемые способы обращения в CSS к атрибутам:

  • div>span - прямая вложенность
  • div, span - или div, или span
  • div span - любая вложенность
  • div+span - span идет сразу после div
  • есть еще другие способы
Свойство Значение Описание
Стилизация текста
font-family имя шрифта задает список шрифтов
font-style normal, italic, oblique нормальный шрифт, курсив, наклонный шрифт
font-size px, pt, em, vw, vh, % размер шрифта (em = 4/3px, vw и vh - размеры в процентном соотношении от ширины и высоты экрана соответственно)
font-weight normal, lighter, bold, bolder, 100–900 жирность шрифта
color green, #код(rgb), hsl(0, 100%, 50%), hsla(0, 100%, 50%, 0.5) задается цвет шрифта
text-decoration none, overline, underline, line throw делает текст подчеркнутым, надчеркнутым, зачеркнутым
text-indent px, pt, em абзац
line-height px, pt, em высота строки (текст расположен по средине строки)
letter-spacing px, pt, em расстояние между буквами
word-spacing px, pt, em расстояние между словами
text-shadow x y r цвет затенение текста(кордината по Х, координата по У, радиус затенения, цвет)
Стилизация фона
background-color green, #код(rgb), hsl(0, 100%, 50%), hsla(0, 100%, 50%, 0.5) задается цвет фона
background-image url(адрес) задается фон, как картинка
background-position x y задается положение фона по горизонтали и по вертикали
background-size px, %, cover, contain задается размер фона(по горизонтали и по вертикали, cover - масштабирует изображение с сохранением пропорций так, чтобы его ширина или высота равнялась ширине или высоте блока, contain - масштабирует изображение с сохранением пропорций таким образом, чтобы картинка целиком поместилась внутрь блока)
background-repeat repeat-x, repeat-y, no-repeat определяет, как будет повторяться фоновое изображение, установленное с помощью свойства background-image
background-attachment scroll, fixed устанавливает, будет ли прокручиваться фоновое изображение вместе с содержимым элемента
