Bootstrap
Есть три способа установки и внедрения Bootstrap в проект:
  1. Использование «npm», запустив команду «npm install bootstrap».
  2. Использование «CDN», вставляя ссылку в проект «link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/...».
  3. Скачивание библиотеки Bootstrap и ее локальное использование.
Bootstrap помогает быстро создавать и настраивайть адаптивные сайты, ориентированные на мобильные устройства, с помощью Bootstrap, самого популярного в мире инструментария с открытым исходным кодом, включающего переменные и миксины Sass, адаптивную сеточную систему (сетка разделена на 12 столбцов), обширные готовые компоненты и мощные плагины JavaScript. Подробнее все описано в документации https://getbootstrap.com/.
